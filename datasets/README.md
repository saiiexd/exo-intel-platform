# ExoIntel Scientific Dataset Documentation

This directory maintains the versioned snapshots of the exoplanet datasets used across the research platform.

## Dataset Stages

1.  **Raw (`planets`)**: Direct ingestion from NASA Exoplanet Archive TAP service.
2.  **Cleaned (`planets_cleaned`)**: Post-diagnostic datasets with physical outlier removal (Z-score > 3).
3.  **Enriched (`planets_enriched`)**: Final research-grade dataset with engineered astrophysical features.

## Engineered Feature Definitions

| Feature | Scientific Description | Motivation |
| :--- | :--- | :--- |
| **Earth Similarity Score (ESS)** | Geometric mean of normalized planetary radius and mass relative to Earth values. | Approximates physical similarity to terrestrial planets. |
| **Stellar Habitability Factor** | Composite metric of host star luminosity and effective temperature relative to the Solar standard. | Evaluates the parent star's suitability for life-hosting conditions. |
| **Density Ratio** | Ratio of planetary density to Earth's bulk density (5.51 g/cm³). | Identifies rocky vs. gaseous composition. |
| **ML Habitability Index** | Predictive score [0, 1] generated by the Gradient Boosting model. | Synthesizes complex multi-variate correlations for habitability. |

## Versioning Policy
Datasets are snapshotted during full pipeline runs to ensure research reproducibility. Artifacts are stored as `.parquet` or `.csv` in versioned sub-directories.
