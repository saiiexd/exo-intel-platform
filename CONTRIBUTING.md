# Contributing to ExoIntel

First off, thank you for considering contributing to ExoIntel! It's people like you that make ExoIntel such a great scientific platform.

## Code of Conduct
This project and everyone participating in it is governed by the [ExoIntel Code of Conduct](CODE_OF_CONDUCT.md). By participating, you are expected to uphold this code.

## How Can I Contribute?

### Reporting Bugs
If you find a bug in the source code, you can help us by submitting an issue to our GitHub Repository. Even better, you can submit a Pull Request with a fix.

### Suggesting Enhancements
If you have an idea for an enhancement, an astrophysics feature engineering improvement, or a new ML technique, please submit an issue explaining your rationale. 

### Pull Requests
1. Fork the repo and create your branch from `main`.
2. Ensure you have activated the virtual environment and installed all dependencies from `requirements.txt` and `package.json`.
3. If you've added code that should be tested, add tests.
4. If you've changed APIs, update the documentation.
5. Ensure the test suite passes on both the frontend and backend.
6. Issue that pull request!

## Setting up your environment
Please refer to the `README.md` and `community/development_setup.md` for specific instructions on building the platform locally.

**Thank you for your scientific contributions!**
